# This code is based on Android Studio

## Environment
- Android Studio
- Target Device Version: API level 34

## Each folder role
- drawable: 앱 버튼 이미지, 사용예시 설명
- layout: 앱 구성화면
- myapplication: 앱 기능 코드 (블루투스 연결, 센서값 처리)
